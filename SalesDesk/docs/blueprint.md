# **App Name**: SalesDesk

## Core Features:

- Secure User Authentication: System for user login with distinct roles for Administrator and Seller. Includes password reset functionality for sellers by administrators.
- Product & Commission Management: Administrators can create, edit, and delete products (chairs and tables), assign prices, and define default commission rates (default $200, editable per sale).
- Seller & Inventory Control: Administrators manage seller accounts, assign product inventory to each seller, and manually assign sales. Automatic inventory deduction upon sale confirmation.
- Sales Transaction Processing: Registers sales, automatically calculates total_venta, total_comision, and total_deposito, freezing these values at the time of sale. Sellers confirm delivery of their assigned sales.
- Administrator Performance Dashboard: Admins view weekly sales reports per seller, total sales, total commissions, and total deposits. Includes filters for week, seller, city, and sales status (assigned, delivered, paid) with graphical representation.
- Seller's Personal Overview: Sellers view their assigned inventory, allocated sales, weekly sales performance, total commission earned, and the total amount they need to deposit.
- AI-powered Sales Insight Tool: An administrative tool that analyzes sales data to identify trends and suggest optimal commission rate adjustments or pricing strategies to enhance seller performance and revenue.

## Style Guidelines:

- Primary color: A deep, professional blue (#1F64B2) to evoke trust and reliability for key interactive elements and branding.
- Background color: A subtle, desaturated blue (#ECF0F4) providing a clean, bright canvas that complements the primary color.
- Accent color: A bright, contrasting turquoise (#70DBDB) used sparingly for highlights, call-to-action elements, and chart differentiation.
- Body and headline font: 'Inter' (sans-serif) for its modern, clear, and objective aesthetic, ensuring readability across all data-intensive interfaces.
- Clean, consistent, and minimalistic linear icons to ensure clarity and improve navigation within both administrator and seller panels.
- A spacious, responsive, grid-based layout featuring distinct navigation sidebars and clear, card-based summaries for efficient information display, tailored for professional data presentation in admin and simplified views for sellers.
- Subtle, smooth transitions for UI element states, navigation changes, and dashboard interactions to provide a polished and modern user experience.
